Widget functionality test page

Use this page to verify every AccessShield widget feature. Open the blue accessibility launcher (bottom-right), then work through the checklist below.

Testing checklist

  • Visual: font size, dyslexia font, dark/light mode, contrast, grayscale, saturation
  • Reading: reading guide, reading mask, link highlight, focus indicator
  • Speech: read selection, read page, click-to-read, speed (slow/normal/fast), stop
  • Navigation: keyboard nav mode, skip link, focus tracker
  • Language: switch widget UI to Hindi (हिन्दी)
  • Reset: “Reset All” restores defaults and clears host-page changes
